# Laberinto

## Descricion

Mickey Mouse esta atrapado en un laberinto y deberas ayudarlo a escapar.

Usaremos estos caracteres para el laberinto para el laberinto

```
@ Mickey
# Para los obstaculos
' ' (Whitespace) Para las casillas libres
E Este para la salida
```
